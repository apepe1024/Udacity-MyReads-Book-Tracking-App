# Udacity My Reads App

This is the mid-way project for the "Building With React" section of the Udacity Front-End Web Developer Nanodegree Program.

## Getting Started

### Install
```
$ npm install
```
### Run App
```
$ npm start
```
### View application
```
http://localhost:3000
```

## Prerequisites

* [Node.js](https://nodejs.org/en/)
* [npm](https://www.npmjs.com/)
